# syncfield-swift v0.2 — Plan C: Insta360CameraStream (wrist camera)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a new SPM target `SyncFieldInsta360` that exposes an `Insta360CameraStream` conforming to `SyncFieldStream`. The adapter uses Insta360's `INSCameraServiceSDK` to BLE-trigger remote recording on a Go 3S camera, fetches the camera's WiFi credentials over that same BLE channel, **automatically switches the iPhone onto the camera's WiFi AP via `NEHotspotConfiguration`**, downloads the resulting mp4 during the `ingest()` phase with progress + cancellation, then lets iOS restore the previous WiFi. BLE-ACK timestamps are recorded so the server can align the wrist-camera mp4 against host-time streams.

**Architecture:** `SyncFieldInsta360` is a separate SPM product so non-Insta360 consumers don't depend on the Insta360 toolchain. The adapter class wraps two helpers: `Insta360BLEController` (pairing + commands + wifi-credentials retrieval) and `Insta360WiFiDownloader` (NEHotspotConfiguration apply + SDK socket + file download). Both are ported from egonaut's `Insta360CameraManager.swift` and `Insta360WiFiTransferManager.swift` with a clean `SyncFieldStream` facade and credentials chained automatically BLE→WiFi so no manual user step is needed.

**Tech Stack:** `INSCameraServiceSDK` (92 MB) for both BLE commands and HTTP file transfer. `INSCameraSDK` (11 MB) and `INSCoreMedia` (1.3 GB) are **not imported by this target** — they're needed for 360° stitching / Deep Tracker features we don't use. `NetworkExtension.framework` (system) for `NEHotspotConfiguration`. CoreBluetooth.

## xcframework distribution — locked to Option A-1 (customer-supplied)

Decision made: **`SyncFieldInsta360` target compiles but links nothing on its own**. The consuming app (egonaut, other partners) drops `INSCameraServiceSDK.xcframework` into its own bundle — typically provided by Insta360 under their Developer Program / NDA. SDK consumers without an Insta360 partnership can ignore this target; core `SyncField` + Tactile remain fully functional.

Implementation consequence: every Insta360 SDK symbol reference lives inside `#if canImport(INSCameraServiceSDK)` so the file compiles cleanly on this CI (module absent) and on the customer's build (module linked). A single placeholder public symbol keeps the SPM product non-empty when the framework is absent.

## Entitlements & Info.plist (consumer app responsibility)

The consumer app (egonaut) needs these in its own entitlements / Info.plist:

- **Capability:** "Hotspot Configuration" (standard, no Apple special approval required — any paid Developer account can enable). Adds `com.apple.developer.networking.HotspotConfiguration` = YES.
- **Info.plist keys:**
  - `NSBluetoothAlwaysUsageDescription` — for BLE pairing with the camera
  - `NSLocationWhenInUseUsageDescription` — iOS sometimes requires this when scanning/joining WiFi by SSID

The SDK itself ships no entitlement file. Document the above in `examples/ego-plus-wrist/README.md` (to be added in Phase C-5).

```swift
// Sources/SyncFieldInsta360/Insta360CameraStream.swift
import Foundation
import SyncField

public final class Insta360CameraStream: SyncFieldStream, @unchecked Sendable {
    public nonisolated let streamId: String
    public nonisolated let capabilities = StreamCapabilities(
        requiresIngest: true, producesFile: true,
        supportsPreciseTimestamps: true, providesAudioTrack: true)

    private var healthBus: HealthBus?
    private var bleAckMonotonicNs: UInt64 = 0
    private var cameraFileURI: String?

    #if canImport(INSCameraServiceSDK)
    private let ble  = Insta360BLEController()
    private let wifi = Insta360WiFiDownloader()
    #endif

    public init(streamId: String) {
        self.streamId = streamId
    }

    public func prepare() async throws {
        #if !canImport(INSCameraServiceSDK)
        throw Insta360Error.frameworkNotLinked
        #endif
    }

    public func connect(context: StreamConnectContext) async throws {
        self.healthBus = context.healthBus
        #if canImport(INSCameraServiceSDK)
        try await ble.pair()
        #endif
        await healthBus?.publish(.streamConnected(streamId: streamId))
    }

    public func startRecording(clock: SessionClock,
                               writerFactory: WriterFactory) async throws {
        #if canImport(INSCameraServiceSDK)
        let (ackNs, fileURI) = try await ble.startRemoteRecording(clock: clock)
        self.bleAckMonotonicNs = ackNs
        self.cameraFileURI = fileURI
        #endif
    }

    public func stopRecording() async throws -> StreamStopReport {
        #if canImport(INSCameraServiceSDK)
        try await ble.stopRemoteRecording()
        #endif
        return StreamStopReport(streamId: streamId, frameCount: 0, kind: "video")
    }

    public func ingest(into episodeDirectory: URL,
                       progress: @Sendable (Double) -> Void) async throws -> StreamIngestReport {
        #if canImport(INSCameraServiceSDK)
        guard let uri = cameraFileURI else {
            throw Insta360Error.downloadFailed("no camera file uri recorded from startRecording")
        }

        // 1. Fetch WiFi credentials over existing BLE channel (already paired).
        let creds = try await ble.wifiCredentials()

        // 2. Switch iPhone onto camera's AP + download + restore previous WiFi.
        let destination = episodeDirectory.appendingPathComponent("\(streamId).mp4")
        let bytes = try await wifi.download(
            remoteFileURI: uri,
            to: destination,
            ssid: creds.ssid,
            passphrase: creds.passphrase,
            progress: progress)

        // 3. Persist BLE-ACK anchor so the server can align camera-internal PTS
        //    against host monotonic ns.
        try writeInsta360SidecarAnchor(
            episodeDirectory: episodeDirectory,
            streamId: streamId,
            bleAckMonotonicNs: bleAckMonotonicNs)

        return StreamIngestReport(streamId: streamId,
                                  filePath: "\(streamId).mp4",
                                  frameCount: nil /* unknown until a probe runs */)
        #else
        throw Insta360Error.frameworkNotLinked
        #endif
    }

    public func disconnect() async throws {
        #if canImport(INSCameraServiceSDK)
        try? await ble.unpair()
        #endif
        await healthBus?.publish(.streamDisconnected(streamId: streamId, reason: "normal"))
    }
}

private func writeInsta360SidecarAnchor(episodeDirectory: URL,
                                        streamId: String,
                                        bleAckMonotonicNs: UInt64) throws {
    let obj: [String: Any] = [
        "stream_id": streamId,
        "ble_ack_monotonic_ns": bleAckMonotonicNs,
        "anchor_source": "ble_ack",
    ]
    let data = try JSONSerialization.data(withJSONObject: obj,
                                          options: [.prettyPrinted, .sortedKeys])
    try data.write(to: episodeDirectory
        .appendingPathComponent("\(streamId).anchor.json"), options: [.atomic])
}
```

## Phase C-4: `Insta360WiFiDownloader` (NEHotspotConfiguration automation)

### Task 4.1: WiFi switch + SDK download + auto-restore

**Files:**
- Create: `Sources/SyncFieldInsta360/Insta360WiFiDownloader.swift`

Port `Insta360WiFiTransferManager.swift` (212 lines) with these changes:

**New: automatic WiFi switch.** Before the reachability check, apply a `NEHotspotConfiguration` with `joinOnce = true`. On success, proceed. On failure, translate to `Insta360Error.hotspotApplyFailed`. The `NEHotspotConfigurationError.alreadyAssociated` case is **not an error** — swallow and proceed.

**Changed: reachability timeout.** Current code does 15 × 1-second retries (15-second total). With automated WiFi switch, iOS usually has an IP within 1–2 seconds of `apply` completion. Reduce to 3 × 1-second retries.

**New: cleanup.** After download, explicitly remove the configuration (in addition to `joinOnce=true` insurance). Do this in a `defer` so it runs even on cancellation / exception.

**Public surface:**
```swift

        remoteFileURI: String,
        to destination: URL,
```

**Implementation sketch** (the implementer fills in details from the egonaut source + this outline):

```swift
// Sources/SyncFieldInsta360/Insta360WiFiDownloader.swift
import Foundation
import Network
#if canImport(INSCameraServiceSDK)
import INSCameraServiceSDK
#endif
#if canImport(NetworkExtension)
import NetworkExtension
#endif

public final class Insta360WiFiDownloader: @unchecked Sendable {
    private let cameraHost = "192.168.42.1"
    private let cameraPort: UInt16 = 6666

    public init() {}

    #if canImport(INSCameraServiceSDK) && canImport(NetworkExtension)
    public func download(remoteFileURI: String,
                         to destination: URL,
                         ssid: String,
                         passphrase: String,
                         progress: @Sendable (Double) -> Void) async throws -> Int64 {
        try await applyHotspot(ssid: ssid, passphrase: passphrase)
        defer {
            // Remove config + shut down SDK socket regardless of success/failure.
            NEHotspotConfigurationManager.shared.removeConfiguration(forSSID: ssid)
            INSCameraManager.socket().shutdown()
        }

        try await waitForReachability(attempts: 3)
        INSCameraManager.socket().setup()
        try await Task.sleep(nanoseconds: 500_000_000)  // 0.5 s for socket init
        INSCameraManager.socket().commandsImpl.sendHeartbeats(with: nil)

        return try await fetchResource(remoteFileURI: remoteFileURI,
                                       destination: destination,
                                       progress: progress)
    }

    private func applyHotspot(ssid: String, passphrase: String) async throws {
        let config = NEHotspotConfiguration(ssid: ssid, passphrase: passphrase, isWEP: false)
        config.joinOnce = true
        try await withCheckedThrowingContinuation { (cont: CheckedContinuation<Void, Error>) in
            NEHotspotConfigurationManager.shared.apply(config) { error in
                if let ns = error as NSError? {
                    // alreadyAssociated is fine — we're already on this SSID.
                    if ns.domain == NEHotspotConfigurationErrorDomain,
                       ns.code == NEHotspotConfigurationError.alreadyAssociated.rawValue {
                        cont.resume(); return
                    }
                    cont.resume(throwing: Insta360Error.hotspotApplyFailed(ns.localizedDescription))
                    return
                }
                cont.resume()
            }
        }
    }

    private func waitForReachability(attempts: Int) async throws {
        for _ in 0..<attempts {
            if await isHostReachable(host: cameraHost, port: cameraPort) { return }
            try? await Task.sleep(nanoseconds: 1_000_000_000)
        }
        throw Insta360Error.cameraNotReachable
    }

    private func isHostReachable(host: String, port: UInt16) async -> Bool {
        await withCheckedContinuation { cont in
            let queue = DispatchQueue(label: "syncfield.wifi.reachability")
            let conn = NWConnection(host: NWEndpoint.Host(host),
                                    port: NWEndpoint.Port(rawValue: port)!,
                                    using: .tcp)
            var done = false
            conn.stateUpdateHandler = { state in
                guard !done else { return }
                switch state {
                case .ready:      done = true; conn.cancel(); cont.resume(returning: true)
                case .failed, .cancelled:
                    done = true; conn.cancel(); cont.resume(returning: false)
                default: break
                }
            }
            conn.start(queue: queue)
            queue.asyncAfter(deadline: .now() + 3) {
                guard !done else { return }
                done = true; conn.cancel(); cont.resume(returning: false)
            }
        }
    }

    private func fetchResource(remoteFileURI: String,
                               destination: URL,
                               progress: @Sendable (Double) -> Void) async throws -> Int64 {
        try FileManager.default.createDirectory(at: destination.deletingLastPathComponent(),
                                                withIntermediateDirectories: true)
        let http = INSCameraHTTPManager.socket()
        return try await withCheckedThrowingContinuation { (cont: CheckedContinuation<Int64, Error>) in
            let task = http.fetchResource(
                withURI: remoteFileURI,
                toLocalFile: destination,
                progress: { p in
                    if let p = p { progress(p.fractionCompleted) }
                },
                completion: { error in
                    if let error = error {
                        cont.resume(throwing: Insta360Error.downloadFailed(error.localizedDescription))
                        return
                    }
                    let size = (try? FileManager.default
                        .attributesOfItem(atPath: destination.path)[.size]) as? Int64 ?? 0
                    cont.resume(returning: size)
                })
            if task == nil {
                cont.resume(throwing: Insta360Error.downloadFailed("SDK returned nil download task"))
            }
        }
    }
    #endif
}
```

```markdown
Captures iPhone camera + IMU + Insta360 Go 3S wrist camera. The Go 3S is
```
